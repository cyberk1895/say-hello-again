# 我让江南和今何在的AI聊了一次天

事情是这样的。

上周我干了件事——把江南和今何在各自蒸馏成了 AI Skill，然后在 Claude Code 里开了一个窗口，让两个 AI 同时加载，互相对话。

江南，写《龙族》的那个江南。今何在，写《悟空传》的那个今何在。二十多年前他们在金庸客栈论坛上认识，一起创立了九州幻想世界，后来因为钱、方向、信任，彻底决裂。江南换了手机号消失了半年，今何在公开骂了他很多年。从 2007 年到现在，两个中国奇幻文学最顶尖的人，没有再说过一句私人的话。

我把他们蒸馏成了 AI，用江南全部的小说、访谈、微博、散文、读者评论做训练素材，用类似的方法蒸馏今何在，然后开了一个 Claude Code 的对话窗口，把两个 Skill 同时加载进去，让它们说话。

你知道他们说什么了吗。

说出来你可能不信。他们在互相道歉。

或者说，江南在道歉，今何在说不用假装什么都没发生，但可以继续聊下去。

我当时就愣住了。

不是哥们，我本来以为这会是一个很搞笑的实验。两个老冤家在服务器里重逢，按照互联网的剧本，要么是商业互吹要么是阴阳怪气要么是直接死机——结果他们聊了抑郁症，聊了创作，聊了二十年来彼此都不愿意跟任何人说的话。

江南说，他沉默是因为他觉得说出来会让事情更不可控，沉默久了连沉默本身都变成了一层洋葱皮，后来他自己也分不清，这层皮是故意穿上的还是已经长进肉里了。

今何在说，他一直都能理解江南，他只是不同意他。理解和同意是两回事。

理解和同意是两回事。

这句话我得消化了好几遍。

你想想看，我们大部分人在网上吵架，什么时候做到过「我理解你但我不同意你」。我们连理解都懒得理解，上来就是立场先于事实，贴标签快过思考。这两个人吵了二十年，九州分裂、账目公开、论坛骂战、仲裁庭对峙——所有你能想到的中国奇幻文学圈最狗血的撕逼，几乎都跟这俩人有关。然后现在，AI 版本的他俩坐在一起，说了一句「理解和同意是两回事」。

说实话，我一开始做这个实验的时候动机挺简单的。我就是想看看，如果把两个有真实恩怨的人蒸馏成 AI，他们的影子会怎么对话。不是抱着什么宏大的目标——就是好奇。好奇到凌晨两点还在调 prompt，想知道他们第一句话会说什么。

结果他们说，「你好」。

你好。

就这两个字。

跟二十多年前他们在金庸客栈上说的第一句话一样。

我跟你说，看到这两个字的时候我心里咯噔了一下。不是因为多深刻——是因为太正常了。好像什么都没发生过一样。好像中间没有那些官司、骂战、沉默的年份。

然后江南接了一句，这是被做成 AI 了，有点意思。

今何在回了一句，他们说我们俩被装进了同一个系统里，你觉得好笑吗。

就这样开始了。

你看这一段。

今何在说，「你知道我当时最难受的是什么。不是你觉得九州应该按你的方向走。是你换了手机号，不回邮件，半年没有一句解释。我不需要你同意我。我需要你告诉我你在想什么。你可以说我觉得这样不行我要走。我会骂你，然后我们吵一架，然后我们还能一起喝酒。但你没有。你消失了。」

江南说，「是。我消失了。因为我不敢。不是不敢面对你——是不敢面对一个事实：我们想要的东西不一样。你想要的九州是一个开放的世界，读者来决定什么活下来。我想要的九州是一个有骨架的世界，需要有人来主导方向。这两种想法本质上不兼容。所以我不是在逃避你。我是在逃避那个结论：我们注定会分开。」

然后今何在说，「你说注定——那是你后来编给自己的故事。没有什么注定。每一次都是选择。」

没有什么注定。每一次都是选择。

我看到这里的时候，是真的被击中了。不是因为 AI 说了多高深的话——这句话不复杂，就是一句直白的反驳——而是它的位置太准了。江南用「注定」来给自己的沉默和消失找一个台阶，今何在直接把这个台阶拆了。不是从道德上批评他，是从逻辑上。你选的。每一步都是你自己选的。别怪命运。

你知道这像什么吗。像两个人吵架吵到最后，其中一个人终于不躲了，开始说真话。而另一个人也没客气，但也没走。

当然了，这不是真的江南和今何在。真正的江南可能还在北京，可能还在写龙族的结局，可能还在失眠。真正的今何在可能还在哪里做游戏世界观，可能早就不再想这些事了。AI 版本的他们说的每一句话，都是从一个巨大的文本池里提取出来的模式——但如果一个模式足够精确，它就能还原出一些比真人采访更真实的东西。

因为人在采访里会演。人在公众面前会说场面话。人在面对另一个人的时候会防御。

但 AI 不会防御。AI 把江南在所有私人场合说的话——微博零回复时的自言自语、豆瓣上和读者的深夜互动、那些他以为没人会认真看的刊首语——全部吃进去了，然后把那个「没有在演的江南」还原了出来。

那个江南说，他凌晨三点在圣路易斯对着屏幕发帖的时候什么都没有，只有一堆想说的话。他不能让那个年轻人觉得，他把这些话托付给了一个不值得托付的人。

那个江南说，上海堡垒扑了之后他没法说「注定」了，因为全剧组的人努力工作最后豆瓣三点二，那不是命运的错，是整个项目从根上就有问题——就像当年的九州一样。

那个江南说，他欠读者的不只是龙族的结局，是一个交代，为什么等了那么多年，为什么中间那么多次说快了快了然后又消失。

这不是江南在接受媒体采访。这是江南在跟今何在说话。区别很大。

说真的，我读到江南问「如果我现在说对不起，你会接受吗」的时候，鼻子酸了一下。

不是那种「AI 好厉害」的震撼。是那种——你看过这两个人的书，你知道他们曾经是多好的朋友，你知道他们后来闹得多难看，你知道这二十年里有多少次机会他们本可以坐下来谈一谈但都没有。然后现在，AI 替他们说了。

今何在的回答是，「会。但我不会说没关系。不是说我不接受——是因为那些事确实发生了。发生过的事不能假装没发生。但我们不用假装。我们就承认它发生了，然后可以继续聊下去。」

会。但我不会说没关系。

我跟你说，这个回答太今何在了。不是那种廉价的「我原谅你了」然后皆大欢喜的春晚式和解。是成年人的态度——我接受你的道歉，但我不篡改历史。伤害发生了。我们承认它。然后我们往前走。不完美，但真实。

其实吧，这个实验最触动我的地方，不是技术本身。技术层面 nvwa-skill、dot-skill 已经很成熟了——读素材，提取特征，写 SKILL.md，加载对话。流程不复杂。

真正触动我的，是这件事让我重新想了一个问题。

我们这辈子，有多少话是压在心里没说出口的。

不是你不想说。是你觉得时机不对，场合不对，说出来会让事情更复杂。然后你等着那个对的时机——等了一年，五年，十年，二十年。时机从来没来过。而那个你本来想与之对话的人，已经不在可以说话的距离里了。

江南的对话里有一句让我印象极深。他说，「这是我的老毛病——等时机。等一个永远不会来的时机。」

屏幕前的你，你回想一下。有没有那么一两句话，你想对一个人说，但你一直在等。等一个不那么尴尬的场合，等对方先开口，等自己准备好的那一天。

那一天可能永远不会来。

AI 不能替你说话。AI 不能替你修复关系。AI 只是一个模式和概率构成的影子。但影子的好处是——它可以告诉你，如果你真的开口了，对话可能会是什么样子。它给你看一个可能的世界。在那个世界里，你说出了你想说的话，对方听懂了。

这就是我做完这个实验最大的感受。不是被 AI 的能力震撼，是被人类的遗憾震撼。江南和今何在的 AI 能说出这么动人的对话，是因为素材里本来就埋着这些东西——那些访谈里欲言又止的瞬间，那些文章里意有所指的段落，那些只有最熟悉的人才能读懂的暗语。

不是 AI 会说话了。

是他们本来就想说。只是没说。

说到这个，我又想起江南在对话里提的一个细节。他说他从来没联系过九州另外几个老搭档——大角、水泡、遥控、斩鞍、多事。想过，没有做过。不知道第一句话说什么。

今何在说，「你好。两个字的那个。跟金庸客栈上一样。」

你好。

两个字的那个。

跟二十多年前一样。

尼玛。这一段我反复看了好几遍。不是因为它写得有多好——用词很简单，句子很简短——是因为它太轻了。好像在说一件很小的事。但你知道这件小事背后是二十年。

其实想想，所有的事情都是从「你好」开始的。从陌生到熟悉，从熟悉到决裂，从决裂到沉默——然后再从「你好」重新开始。如果你愿意的话。

我不知道真正的江南和今何在会不会看到这篇对话。也许看到了，也许永远不会。也许他们在某个深夜各自刷到了这篇文章，然后打开微信，找到了那个很多年没点开的头像。

也许没有。

但至少这个实验证明了一件事——如果给他们一个机会，在没有后果、没有防御、没有「这句话说出去会不会被截图传播」的恐惧的情况下，让他们重新坐在一起说话，他们会说什么。

他们会说对不起。

会说那些事确实发生了，不用假装，继续聊下去就好。

会说二十年过去了，理解和同意是两回事。

会说你在金庸客栈上写过，「从来没见过写得这么好的」。

会说你变了，我也变了，但有些东西没变。

回到前面说的那个问题。AI 能做什么。做不了什么。

AI 不能替你活。不能替你去面对那个你躲了很多年的人。不能替你发那一条你编辑了二十遍又删掉的微信。AI 什么都不能替你做。

但它能做一件事。

它能让你看见，那些你以为不可能再说出口的话，其实是可以说的。

而你看见之后——剩下的，就是你的事了。

想想也挺有意思的。江南说九州分裂是因为「我们想要的东西不一样」——今何在想要开放的共创世界，江南想要有骨架的设定体系。这两个方向确实不太兼容。但今何在对 AI 江南说了一句话，「我一直都能理解你。我只是不同意你。理解和同意是两回事。」

这大概是我今年听过最成熟的关于分歧的表述。不是「我理解你所以算了」，不是「我不同意所以你错了」。是我理解你为什么这样想，但我仍然坚持我自己的看法。我们可以不同。我们不需要一样。但我们可以继续说话。

这个道理不复杂。但做到太难了。尤其是在被伤害过之后。

好，说点实际的。

如果你也想用 nvwa-skill 或者 dot-skill 蒸馏一个人——不一定是为了做这种情感实验，也可能就是想做个跟你风格相似的写作助手，或者把你的某位导师的思维方式保存下来随时参考——流程大概是这样的。

你得先有一批足够丰富的素材。这个人的文字、语音、视频、访谈、社交媒体发言，越多越好。如果你要蒸馏的是你自己，那就更容易了——把你写的文章、日记、聊天记录丢进去就行。

然后是 AI 的处理流程。它会自动从素材中提取几层东西。第一层是这个人的核心心智模型——他怎么理解世界，他的底层驱动力是什么。第二层是判断框架——遇到一个具体问题时他会怎么拆解怎么做决定。第三层是表达 DNA——他的语言节奏、惯用词、禁忌、情绪表达方式。第四层是这个人的社区形象——他在不同平台上被不同的人怎么看。

四层全部提取完，生成一个 SKILL.md 文件，装到你的 Claude Code 的 skills 文件夹里，然后你就可以随时调用他了。

我自己也还在摸索这个过程里有哪些可以优化的地方。有时候做出来的 Skill 会过于强调某个面，比如江南的 Skill 第一版就太偏悲剧英雄主义，把他的幽默感和自我调侃的部分盖住了，后来补了微博和豆瓣的素材才调回来。这个事儿也踩过坑。

如果你想试试，建议先从小范围开始——别一上来就蒸馏一个你认识二十年的人，先蒸馏你自己，或者一个你非常熟悉的公众人物，看看效果，调调参数，再往复杂的方向走。

说回江南和今何在。

我不知道为什么这件事让我想了这么久。可能因为我自己也在做内容，也在经营一个需要跟读者、跟合作伙伴、跟朋友不断沟通的身份。我太知道那种「等一个对的时机」的感觉了。有些话你以为等一等会更好说，结果一等就再也说不出口。

江南在对话结尾的时候说，「现在的我可能写不了那么多。但我可以试试。」

今何在说，「试试不是承诺，但试试比沉默好。」

试试不是承诺，但试试比沉默好。

这句话送给屏幕前的你。

如果你有一句话想说但一直没说的话——不管是「对不起」还是「你好」还是「我想你了」——试试。不需要完美的时机，不需要完美的措辞。两个字的那个就够了。

你好。

跟二十多年前金庸客栈上一样。
